Location-driven web solution for promoting tourism in Romania

Problem:
Create packages for Exotic and City break tourism, 
they might change in the future or new ones will be created.
The complexity of the creation of an object is hidden from the client. The objects should
not be created using the constructor, but using the object of the factory type.

Advantages:
All object created have in common the interface
Strict control of instantiaion: 
The objects should not be created using the constructor, but using the object of the factory type.

Disadvantages:
"New" objects cannot be created
the constructors are private, the classes cannot be extended
