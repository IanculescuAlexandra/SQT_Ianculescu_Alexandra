Location-driven web solution for promoting tourism in Romania

Problem:
A large volum of memory is used which makes the programme to run late. 
Share memory between similar objects

Advantages:
It reduces the memory occupied by objects by sharing them between clients
To properly manage the sharing of Flyweight objects between clients they must be immutable

Disadvantages:
The effects are visible for solutions where the number of objects is large
The reduced memory level depends on the number of Flyweigh object categories
